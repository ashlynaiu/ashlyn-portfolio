Custom Sass Framework

Summary

This is a personal style guide project I continuously work. The style guide is a quick, easy to use grid and svg manager. I am constantly adding and updating the styles so that it is slim and light weight framework.

Check out the hosted style guide. The hosted style guide is a heroku app, so it may take a bit of time to start up. Or you can check out the repository for the project.

Framework Deatils

Along with a grid, my framework correctly includes basic element resets and a cool svg manager. Using the svg as a data uri, the framework has a variety of mixins that help place and size the image within the layout. The mixins easily account for mobile resizing as well. Overall, its one of my favorite aspects of the framework.
