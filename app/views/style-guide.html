Style Guide Creation

Summary

This is a personal style guide project I continuously work. The style guide is a quick, easy to use grid and svg manager. I am constantly adding and updating the styles so that it is slim and light wait framework.

Framework Deatils

Along with a grid, my framework correctly includes basic element resets and a cool svg manager. Using the svg as a data uri, the framework has a variety of mixins that help place and size the image within the layout. The mixins easily account for mobile resizing as well. Overall, its one of my favorite aspects of the framework.

Next Steps

Along with better documentation, I am also working on adding form styles that would include validation and animations. I'm also hoping to add additional lightweight components.
