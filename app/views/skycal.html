Skycal Web Application

Summary

Skycal is an event calendar for astronomical events. This web application uses React/Redux, MongoDB, Node, SASS, and Webpack. Check out the work in progress version at skycalapp.herokuapp.com.

The idea came from a need for a good astronomical calendar of events. Most astronomy calendars are very poorly designed or not regularly updated. Since buying a telescope for my significant other, I have decided that solving this need would be a great opportunity to do some fun design and development!

Planning Stage

For this project, I am working with a Full Stack Engineer. Originally, we decided on experimenting with a Python/React/Webpack web application. After realizing our small application would be unnecessarily complicated and heavy using Python, we abandoned it and settled on a React/Redux, MongoDB, Node, SASS, and Webpack application. It was surprising how long it took to just decide on the technologies and frameworks we were going to use with this project.

Wireframing

Starting this design consisted of a quick brainstorming session and wireframing. I worked with the other Engineer to come up with a good game plan before started the high fidelity mockups.

First Mock Ups

My initial thought is that the first view would show upcoming events. Then when the user clicks more, the window would expand to show a more detailed calendar version. Some other fun things I have decided to incorporate into the design is icons representing astronomical events and a nice cool color scheme. The first high fidelity mock ups are below.

A Work in Progress

Through countless trial and error, a couple new repos, and many design changes, Skycal has been coming along nicely. Thanks to imjohsep, a light weight React/Redux application with Webpack and Node was built. We collaborated together on a Github repo to create modular Javascript components that were readable and small. Though the CSS is not entirely modular currently, it was important that we got a working, beautiful version of Skycal public so that we could show it off to our friends and family.

Skycal is also responsive and looks great on mobile devices as well! Personally, I think our web application is much more beautiful on a mobile device. The svg icons and fonts look so sharp on retina devices. Make sure you check it out. skycalapp.herokuapp.com

Next Steps

I'll be working on breaking down our CSS/Sass into more manageable, sensible files. Along with that, I'm throughly excited to start incorporating animations and transitions. React has been exciting to learn and truly powerful to develop with; I'm looking forward to continue to work on Skycal.
